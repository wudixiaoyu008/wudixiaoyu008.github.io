2. WHY WE NEED IT

Is This A Problem?

DSP is an ad platform that helps advertisers create and manage ads campaigns. It's really powerful on execution, but that's not working for planning stage. Since users always only have a rough idea when they come to plan on DSP. HMW bridge the gap?


Let's Dig Deeper

1. Current planners only provide insights at a line level, users are requesting a campaign planner.
2. There are many planners on our DSP which already causes confusion. Not all users know when, why, or how to use the data from given planners.
3. Users currently build the Media Plan outside DSP, like in Excel or Slides, because we don't have the tool.

6. Feature Go Through

Integrate With Line Planners

We hope users could also save Line Plans to Media Plan from Line Planners. We position Media Plans to serve as a repository for line-level insights and connect existing Line Planners. For example:

1. Explore targeting options in Omniscope (Line Planner) and save it to a Media Plan.
2. Create a new Media Plan or save it to an existing Media Plan.
3. Split targeting out to different Line Plans by channel.



Campaign Activation

1. Activate a Media Plan as a campaign. All campaign fields are imported automatically from the Media Plan.
2. Continue to review line level setup and fill in missing fields.
3. Campaign is activated successfully. Users can view the campaign directly or remain in the Media Planner.



Export Media Plan

1. Cuts down on tedious busywork. No more manually putting together Media Plans.
2. We support two export formats. .xls for speadsheet broken by line. .pdf for pitch decks with data vis.
3. Customizable. Choose what lines, columns, or charts are included in the export.

7. What's In The End

MVP Launched & Leading 2.0


Now the MVP was launched. It's simple while still effective for users, they can create a multi-line campaign right in a Line Planner (Omniscope).

At the same time, moving towards to our long term vision, we are working on the 2.0 to leverage our existing planners by serving as an aggregation tool for planner insights. Users will be able to explore in Line Planners and save insights to a Media Plan. The Media Plan serves instead as a repository and blueprint for campaign ideation.
